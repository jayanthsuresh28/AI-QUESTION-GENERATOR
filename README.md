# AI-QUESTION-GENERATOR
This project uses Natural Language Processing to automatically generate 1-mark, 2-mark and higher-mark exam questions from syllabus PDFs. The system analyzes content, extracts key concepts, and creates exam-oriented questions based on selected marks and difficulty, helping students and teachers save time and improve exam preparation.
##Objectives
-Automatically extract text from syllabus/notes PDFs
-Identify important concepts using NLP
-Generate exam-oriented questions
-Reduce manual effort in question paper preparation
##Technologies Used
-Python
-Natural Language Processing (NLP)
-Streamlit (Web UI)
-spaCy, NLTK
-PyMuPDF (PDF text extraction)
-Scikit-learn
##System Features
-Upload syllabus or notes in PDF format
-Select question type (1-mark, 2-mark, 5-mark, 8-mark)
-Choose difficulty level
-Generate questions instantly on a web page
-Simple and user-friendly interface
##System Architecture
-PDF Upload
-Text Extraction 
-Text Cleaning & Concept Extraction
-Question Generation using NLP templates
-Display results on web interface
